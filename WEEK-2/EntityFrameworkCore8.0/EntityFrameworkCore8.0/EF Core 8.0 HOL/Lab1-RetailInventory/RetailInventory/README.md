# Entity Framework Core 8.0 HOL

## Lab 1: Understanding ORM with a Retail Inventory System

### 📌 Objective
This lab demonstrates the basics of **Object Relational Mapping (ORM)** using **Entity Framework Core 8.0**. A simple Retail Inventory Management application is created to understand how C# classes map to SQL Server database tables.

### 📚 Concepts Covered
- What is ORM (Object Relational Mapping)
- Entity Framework Core 8.0
- EF Core vs EF Framework
- Creating a .NET Console Application
- Installing Entity Framework Core Packages
- Creating Entity Models
- Configuring DbContext
- SQL Server Connection

### 🛠 Technologies Used
- C#
- .NET 8 Console Application
- Entity Framework Core 8.0
- SQL Server

### 📁 Project Structure

```
RetailInventory
│
├── Data
│   └── AppDbContext.cs
│
├── Models
│   ├── Product.cs
│   └── Category.cs
│
├── Program.cs
└── RetailInventory.csproj
```

### ✅ Outcome
- Created a Retail Inventory Console Application.
- Configured Entity Framework Core 8.0.
- Created Product and Category entity classes.
- Configured the Application DbContext.
- Connected the project with SQL Server.
